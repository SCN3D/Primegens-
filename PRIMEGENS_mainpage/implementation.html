Implementation Details
PRIMEGENS software is written in ANSI C language with command line executable and also as web server. The program uses two third party executables like Primer3 and Megablast. It has been tested on Unix/Linux and Windows platform. Although it allows multiple input formats for query sequence, we have tested only FASTA format so far. The program uses a configuration file, where user can specify various inputs, primer design specifications and algorithmic parameters.

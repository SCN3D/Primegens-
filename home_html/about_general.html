Regular primer design using PRIMEGENS.

It integrates Primer3 and BLAST in a pipeline to automate the design of high-quality primers and probes using various efficient algorithms. There are three different algorithms: 1) Sequence-Specific Primer Design (SSPD), 2) Fragment-Specific Primer Design (FSPD), and 3) Probe-Specific Primer Design (PSPD). along with probe design capacity, simply called Probes Design algorithm. For this, it uses query file having the sequence for which primers/probes need to be designed, and the database file having all the sequences that are present in the PCR reaction. Sequences in the database files are used to check for any potential cross-hybridization and thereby select primer/probes that are specific to the query sequence. The user can upload their own customized database file or use available genomes supported by the server. We do not recommend user to upload large database like any genome.
